# ipn-studio
=======
**IPN-STUDIO** is a professional network calculation tool designed to simplify IP addressing planning and analysis. Built to meet the needs of network engineers and IT technicians, it provides instant visualization of subnets (VLSM), address ranges, and binary conversions.

## Features

* **Dynamic VLSM Calculator:** Accurately calculates subnet divisions based on CIDR input.
* **Interactive Binary Visualization:** Displays IP address octets and subnet masks in an intuitive binary format.
* **Network Architecture Analysis:** Automatically calculates the network address, broadcast address, and usable host ranges.
* **Modern User Interface:** A clean, dark, and responsive design optimized for intensive use in server rooms or office environments.

## Tech Stack

* **Framework:** Laravel & Vue.js (TypeScript)
* **Styling:** Tailwind CSS
* **Build Tool:** Vite

## Installation & Deployment
git clone https://github.com/AdrNazifWadja-networking/ipn-studio.git
cd ipn-studio
npm install
npm run dev


This project is designed to be easy to deploy. To run it locally:

## Pour exécuter IPN-STUDIO en local sur Windows, assurez-vous d'avoir installé Git pour Windows et Node.js.
1-Cloner le projet : Ouvrez un terminal (Git Bash ou PowerShell) et exécutez :
git clone https://github.com/AdrNazifWadja-networking/ipn-studio.git
cd ipn-studio

2-Installer les dépendances :
npm install
npm run dev

3-L'application sera accessible via l'URL indiquée dans votre terminal généralement http://localhost:5173
